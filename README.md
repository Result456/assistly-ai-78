# AI Productivity Assistant

AI-Powered Workplace Productivity Assistant for ASA_2 Programme

---

## 📌 Project Overview

This is an AI-powered workplace assistant designed to automate productivity tasks such as writing emails, summarising meetings, planning tasks, and interacting with a chatbot.

---

## 🚀 Features

* Smart Email Generator
  Generate professional emails with different tones (formal, friendly, persuasive)

* Meeting Notes Summarizer
  Summarises long notes into key points, action items, and deadlines

* AI Task Planner
  Creates daily or weekly schedules and prioritises tasks

* AI Chatbot Assistant
  Interactive assistant for workplace-related questions

---

* Lovable AI
* React
* Tailwind CSS
